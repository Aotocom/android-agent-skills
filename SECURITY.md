# Security Policy

If you find a security issue in this repository, report it privately first.

- Contact: `krutkjain88@gmail.com`
- Include the affected skill or file, clear reproduction steps, impact, and whether credentials, signing material, or release workflows are involved.
- Do not post secrets, keys, tokens, private APKs, or internal artifacts in public issues or discussions.

What happens next:

- The report will be reviewed and triaged.
- Impact and affected versions will be confirmed.
- A fix, mitigation, or advisory path will be coordinated before anything is discussed publicly.
